# Shooting
Shootingは自作の2Dゲームです。  
* 言語 C#  
* ライブラリ Xna

# 概要
横方向のシューティングです。  
画面右から迫るエネミーを撃破するのが目的です。  
ウェーブをクリアするとボスが現れます。  
ゲームオーバーになるまで続きます。
